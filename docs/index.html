설치하는 방법

1

DMG 파일 다운로드

위 버튼을 클릭하면 GitHub에서 최신 버전의 B-Side.dmg 파일을 받습니다.

2

Applications 폴더로 드래그

다운로드한 DMG를 열면 B-Side 아이콘이 보입니다. 오른쪽 Applications 폴더 위로 드래그하면 설치 완료입니다.

3

보안 경고 허용 (한 번만)

처음 실행하면 macOS가 "개발자를 확인할 수 없습니다" 경고를 표시합니다. 아래 경로에서 허용해 주세요.

시스템 설정›개인정보 보호 및 보안›아래로 스크롤›"그래도 열기" 클릭

이후에는 경고 없이 바로 실행됩니다.

4

메뉴바에서 확인

화면 맨 위 메뉴바에 단어가 보이면 설치 완료입니다. 우클릭하면 단어장 관리 화면이 열립니다.
